# ibtws
抓取IbTws-美股、港股股票行情, 根据交易时间自动抓取
- Description
    - source: 源代码, 所有基础代码, 需要修改config文件, 安装requirements文件中依赖项
    - docker: docker版本服务, 只需要修改docker compose文件中config文件的位置配置即可
